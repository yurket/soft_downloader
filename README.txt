Программа предназначена для скачки всех версий необходимого софта с сайта http://www.oldapps.com
В данный момент работа с другими сайтами не поддерживается. 
Для запуска должен быть установлен интерпретатор Python версии 2.6, 2.7 (на 3+ не работает, версии младше не тестировались)  

Использование:  sd.py dir_name start_url
                dir_name    - имя директории, в которую загружать файлы
                start_url   - страница, на которой находятся ссылки на ВСЕ версии необходимого софта 
                                (обычно такого вида http://www.oldapps.com/nokia_suite.php)

Пример:
        sd.py chrome http://www.oldapps.com/google_chrome.php
        Программа соберет скачает все версии программы, которые есть на заданной странице в папку chrome (создаст эту папку, если ее нет)
