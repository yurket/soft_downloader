Программа предназначена для скачки всех версий необходимого софта.

Для запуска должен быть установлен интерпретатор Python версии 2.7.x (на 3+ не работает, версии младше не тестировались)  

Использование:  sd.py db_path
                db_path - путь к файлу в формате JSON со списком сайтов, с которых будет софт качаться, и аргументов. пример такого файла - db.json

Пример записи, содержащейся в файле:
{
    "url": "http://www.example.com/interesting_soft",
    "dst_folder": "soft_dir",
    "traits": [".exe"]
}
traits - признаки ссылок, которые нужно собирать. Т.е. в данном примере будут собраны все ссылки с подстрокой .exe и скачаны.
         если, например, "traits": ["aaa", "app="], то со стартовой страницы ("url") будут собраны ссылки с подстрокой   
         aaa, затем на каждой из только-что собранных, будут собраны ссылки с подстрокой bbb и загружены в папку dst_foler.
